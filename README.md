# 계산기🤖

> **Step1: 코드 합치기**
- 배정된 짝궁과 코드를 합칩니다.
- GitHub에서 모둠원 중 한 명의 저장소에서 다른 한 명을 콜라보레이터로 등록하여 진행합니다.
    - 서로 코드를 설명하고 비교하여 더 나은 코드로 합칩니다.
        - 한 캠퍼의 코드로만 합치는 일은 없도록 합니다.
        - 한 모둠원의 코드 비율이 70%를 초과할 수 없습니다.
    - 타입 혹은 메서드 단위로 PR을 보내 병합합니다.
